WebXR pose stream → teleop_dev ws bridge

XR-RAF onFrame 에서 매 frame head/hand pose JSON 송신 (ws). teleop_dev sender (arm + hand) 가 이 ws 의 shared array 를 읽어 UDP 로 로봇 PC 에 송신. cam.enabled=true 면 sender.cam (8014) 의 WS-JPEG 를 수신해 카메라 plane 렌더링 (1~3개, head-locked / world-locked).

URL 쿼리: ?cam_off=1 영상 강제 비활성, ?cam_host= / ?cam_port= cam 서버 위치, ?cam_mode=head_locked|world_locked 고정 모드, ?plane_distance= / ?plane_width= / ?plane_height= plane 크기 override.
